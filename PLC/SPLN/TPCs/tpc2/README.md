## SPLN - TPC2

Neste trabalho para casa, decidi criar **2** ***scripts*** que aceitam um ficheiro de *input* com várias entradas. Uma dessas entradas, escolhida aleatóriamente usando a biblioteca ***Random***, é aceite numa gramática gerada.
- A primeira *script* consiste na geração de frases com um provérbio associado. Este provérbio associado é escolhido aleatoriamente entre os vários do ficheiro input *proverbios*.
  - *$ ./provscript*
- A segunda *script* consiste na geração de uma gramática relativa a quem será responsável pelo diário de bordo e por ser piloto de testes. Para isso é o script recebe como input um número associado a um mês, e é escolhido para cada 2º feira desse mês (dia da aula de SPLN) quem será responsável pelas tarefas mencionadas.
  - *$ ./spln-script 3*
